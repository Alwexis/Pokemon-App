    <p>
        This is a project that I made for my portfolio, it's a Pokedex app that uses the <a href="https://pokeapi.co"
            target="_blank">PokeAPI</a> to get all the data.
        This App was maded using Ionic Framework, Angular and Typescript.
        Currently this App allows you to see all the pokemons, their stats, evolutions, moves, locations, etc., and you
        can also add them to your favorites list so you can see them later even if you're not connected to Wi-Fi.
        I'm currently working on a new and improved version of the App, that adds some typical games using Pokémon as
        the topic (Tic Tac Toe, Who's That Pokémon, Black Jack, etc).
